## Вообще не про редхат

Узнал, что в dot можно несколько раз на одну строчку рисовать стрелочку.  Ну типа не: 

``` dot
x -> y
y -> z
```

а

``` dot
x -> y -> z
```

Мелочь, а прикольно. Ещё бы сахарку для короткой записи нескольких веток, типа:

``` dot
x:
-> y;
-> z;
```

вместо:

``` dot
x -> y
x -> z
```
